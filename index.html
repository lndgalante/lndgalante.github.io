Colectivos de Retiro - Work in Progress

The problem: Retiro Bus Station has an average of 50.000 people/day and to know what buses are actually present you have access only through some crowded monitors.

The solution: a Raspberry Pi 3 using computer vision gets the data from the monitors and a web-app displays the arrivals and departures in real-time.

MercadoLibre M√°s Vendidos - more than 600 users

The problem: MercadoLibre doesn't offer a way to sort products by amount of units sold in descending order. You can only sort by price and relevance.

The solution: a Google Chrome extension that takes all the items, re-sorts them and then inserts them back to the DOM. You can also turn it on/off.

Prettier-rator - more than 700 users

The problem: each Prettier plugin has a different syntax and file extension for every code editor like Sublime Text, Atom, and Visual Studio Code.

The solution: a CLI that takes in an input for each setting in Prettier and creates a file for each plugin so you can share it across your team.

Postal Code

The problem if you need to quickly find the postal code for any address you'll have to fill in ugly forms and insert tons of information.

The solution a simple web app that takes your current coordinates and shows the postal code for that particular location.
